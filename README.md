# Rainfall-Runoff-Calculator-Python-
Project Overview:
\n This script calculates runoff depth based on rainfall input and a selected Curve Number. It also includes an optional visualization that illustrates the rainfall–runoff relationship commonly used in stormwater management and drainage design.
\n Features:
\n
\n-Runoff estimation using the SCS CN Formula
\n-Accepts any rainfall depth(mm)
\n-Clean, reusable calculation function
\n-Optional Rainfall-Runoff plot
\n-Easy to extend into a larger hydrology analysis tool
